# Personal Portfolio Website

Welcome to my **Personal Portfolio Website**, built with **HTML**, **CSS**, and **JavaScript**. This responsive portfolio showcases my skills, projects, and contact details in a user-friendly, mobile-first design. 

## Overview

This is a professional portfolio designed to provide a clean, intuitive user experience. It includes:

- **Responsive Layout:** Optimized for all devices, from mobile phones to desktop computers.
- **Sidebar Navigation:** For easy access to my CV, projects, and contact information.
- **Download CV Button:** Allows users to download my resume in PDF format.
- **Contact Form:** A simple form for potential clients or employers to get in touch.
- **Projects Showcase:** Displaying my previous works and accomplishments.

## Features

- **Mobile-First Design:** Automatically adjusts for different screen sizes using Tailwind CSS.
- **Interactive Sidebar:** Includes links to different sections like CV, projects, and contact details.
- **Download CV:** Button available in the sidebar for users to download my resume.
- **Smooth Scrolling (Future Features):** For seamless navigation between sections.

## Technologies Used

- **HTML:** Provides the structure and content of the website.
- **CSS:** Custom styles to enhance the look and feel of the page.
- **Tailwind CSS:** A utility-first CSS framework used to build a responsive and beautiful design.
- **Font Awesome:** Icons used for easy navigation and to enhance the UI.
